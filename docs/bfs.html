Montículo de fibonacci

Alberto Benavides
  • Agrega nodos de valores enteros
  • Cada valor agregado será la raíz de un árbol (se ponen como hijos del nodo rs de modo ilustrativo)
  • El nodo de valor mínimo es el nodo que "palpita"
  • Eliminar el nodo de valor mínimo reestructura el montículo, de modo que
    1. Todas las raíces se comparan entre sí por pares
    2. Si dos raíces tienen el mismo número de hijos, la que tenga el valor mayor se vuelve hija de la que tenga el valor menor
